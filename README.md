# Instant Movie Finder

## Description
Instant Movie Finder is a smart movie recommendation website that allows users to search for movies by name or mood. It provides movie details, similar movie suggestions, and home page recommendations inspired by OTT platforms.

## Features
- Single search bar for movie name or mood-based input
- Home page recommendations
- Movie details displayed in a popup modal
- Similar movie suggestions based on genre
- OTT-inspired dark UI

## APIs Used
- OMDb API for fetching movie data such as posters, ratings, cast, and plot

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Run
1. Get a free OMDb API key
2. Add the API key in `script.js`
3. Open `index.html` in any modern browser
